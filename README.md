<h2 id="quick-start">Quick Start üöÄ</h2>
<ol>
  <li><strong>Clone the repository</strong>:
    <pre><code>git clone https://github.com/crakcode-hub/android-animations.git</code></pre>
  </li>
  <li><strong>Open the project</strong>:
    <ul>
      <li>Use <strong>Android Studio Bumblebee (2021.1.1)</strong> or newer.</li>
      <li>Navigate to the cloned folder and open the project in Android Studio.</li>
    </ul>
  </li>
  <li><strong>Run the app</strong>:
    <ul>
      <li>Build and run the app on a physical device or an emulator.</li>
      <li>For optimal performance, test on devices/emulators with Android 8.0 (API level 26) or above.</li>
    </ul>
  </li>
</ol>

<h2 id="requirements">Requirements üõ†Ô∏è</h2>
<ul>
  <li><strong>Android Studio Bumblebee (2021.1.1)</strong> or newer</li>
  <li>Android <strong>Gradle Plugin</strong> version <code>7.0.3</code> or higher</li>
  <li>JDK 11</li>
  <li>An emulator or a physical Android device with <strong>Android 8.0 (API level 26)</strong> or higher</li>
</ul>
